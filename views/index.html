FCC Metric-Imperial Converter

Example Usage

Easily convert gallons, miles and pounds to their metric equivalent or vice versa. Usage is easy. Simply use the endpoint /api/convert and then specify what you would like to convert. See below for some examples:

kaput-hip.glitch.me/api/convert?input=2.5mi kaput-hip.glitch.me/api/convert?input=5.5kg kaput-hip.glitch.me/api/convert?input=3/4gal kaput-hip.glitch.me/api/convert?input=7/8km kaput-hip.glitch.me/api/convert?input=12lbs

Example Return

The api call will return the requested conversion in JSON format.
See below for an example return:

{ "initNum":2.5, "initUnit":"mi", "returnNum":4.02335, "returnUnit":"km", "string":"2.5 miles converts to 4.02335 kilometers" }
